# Changelog

本项目所有重要变更记录于此。版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

### 变更

- 完善 README / 商店详情页：新增「快速上手」「快捷键速查」「两种模式对比」「常见问题」，移除占位截图段

## [1.0.0] - 2026-06-10

首个公开发布。

### 新增

- **EPUB 阅读**：解析 EPUB（container → OPF → spine），抽取章节纯文本；兼容 UTF-8 / GBK 编码，容错处理损坏实体与畸形路径
- **装饰器渲染**：正文以只读虚拟文档 + Decoration 叠加显示，零文件痕迹、无 dirty 状态、无保存弹窗
- **两种伪装模式**：
  - 代码模式（`util.py`）——正文作 `#` 注释，穿插带语法高亮的真 Python 代码
  - Markdown 模式（`notes.md`）——正文作纯文本段落，穿插 Markdown 结构
- **老板键**：`Alt+Q` 双向切换隐藏 / 恢复
- **进度记忆**：以「章节 + 字符偏移」持久化，与排版无关；重启后 `Alt+]` 续读，精确停在原页
- **章节跳转**：命令面板选章直达
- **去规整化**（固定种子，翻页骨架稳定）：
  - 行宽抖动（`widthJitter`）——打破右边缘整齐矩形
  - 成段混排（`fakeCodeEvery` / `fakeCodeBlock` / `fakeCodeJitter`）——正文与伪装代码成段交替
  - 注释缩进抖动——打破左侧 `#` 竖线
- **8 项可配置**：伪装模式、颜色、每行字数、每页行数、伪装密度与随机度、自定义伪装片段
